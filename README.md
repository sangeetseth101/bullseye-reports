# BullsEye Brand Blueprint™ — Reports

Hosted brand strategy reports generated by [The Brand CEO](https://thebrandceo.com) · Mallika Malhotra.

Each client report is a standalone page under `reports/` and is shared only via a private link sent directly to that client. This repo intentionally does not maintain a public index or directory of client reports.

## Deployment

Deployed on [Vercel](https://vercel.com) and GitHub Pages. Each push to `main` triggers a new deployment.
